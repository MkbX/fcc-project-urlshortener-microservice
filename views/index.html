URL Shortener Microservice

URL Shortener

API

Returns a shortened version of posted URL.

Visiting /api/shorturl/[short_url] redirects to the original URL.

For example, for https://freecodecamp.org as posted url, short_url is 3.

[Base URL]/api/shorturl/0 redirects to https://freecodecamp.org.

Details

POST /api/shorturl

Request
[Base URL]/api/shorturl
Response
{ "original_url": 'https://freecodecamp.org', "short_url": 0 }
